# Specification
La auditoría evaluó los siguientes perímetros:
- **Arquitectura**: Verificación de compilación del proyecto.
- **Nomenclatura**: Uso de atributos de API y convenciones limpios.
- **Estabilidad Async**: Búsqueda de llamadas síncronas bloqueantes.
