# Fix Integration Tests Unauthorized Issues

## Objective
Analyze and resolve the `401 Unauthorized` errors occurring in the integration test suite (`GesFer.Product.Back.IntegrationTests`).

## Description
A significant portion of integration tests is failing with a `401 Unauthorized` error when executing API requests. This seems to be related to test authentication setup and JWT handling.
We need to:
1. Identify the root cause of the missing or invalid authorization headers.
2. Implement a robust solution, like configuring a default test user/token across all authenticated controllers or fixing the test data setup.
3. Ensure the test suite turns green successfully.

## Scope
- Investigate `GesFer.Product.Back.IntegrationTests`.
- Fix authentication mocking or token generation.
- Ensure all 58 currently failing tests pass.
