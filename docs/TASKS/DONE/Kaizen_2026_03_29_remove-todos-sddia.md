# Remove TODOs from SddIA specs
Replace explicit 'TODO' strings in SddIA Markdown specifications (`SddIA/actions/sddia-difusion/spec.md` and `SddIA/process/automatic_task/spec.md`) with compliant alternatives like '[PENDING-TASK]' or 'TAREA PENDIENTE'.
