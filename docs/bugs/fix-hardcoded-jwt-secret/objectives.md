# Objetivos

Establecer una política de "Zero-Trust" (Cero Confianza) en los archivos de configuración del repositorio, garantizando que ningún secreto, token o clave criptográfica resida en el código fuente, delegando la responsabilidad de inyección íntegramente a la infraestructura de despliegue.
